2 terminal client color

add the following line into .bash_profile: 

export CLICOLOR=1

3 terminal meta key

Terminal -> Preferences -> Keyboard -> Use Option as Meta Key

4 set terminal hostname

scutil --set HostName <hostname>
# scutil: system configuation util

5 correct spelling automatically off

System Preferences -> Keyboard -> Text -> Correct spelling automatically off

6 emacs

6.1 install

brew cask install emacs

6.2 command as meta

C-h v mac-option-modifier -> customize -> super C-h v mac-command-modifier -> customize -> meta

6.3 add elpa repository

add following to init file: 

(setq package-archives '(("gnu" . "https://elpa.gnu.org/packages/")
                         ("melpa" . "https://melpa.org/packages/")))

6.4 python ide

6.4.1 install jedi, flake8

pip install flake8 jedi

6.5 pep8

(add-hook 'elpy-mode-hook 'py-autopep8-enable-on-save)

6.6 htmlize

need in org export to html with source code part M-x package-install htmlize

6.7 latex

M-x package-install RET auctex RET M-x package-install RET auctex-latexmk RET

11 anaconda

Do not use conda command, use pip command. the repository with pip is much richer than that of conda. There is one problem: the package install with pip will not be seen in the anaconda-navigator windows. But this do not effect the use of the package.
